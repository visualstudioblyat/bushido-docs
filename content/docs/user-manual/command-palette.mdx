The command palette is the fastest way to get anywhere in the browser. Open it with `Ctrl+K`.

## What It Searches

Start typing and it fuzzy-matches across:

- **Open tabs** — switch to any tab by title or URL
- **Bookmarks** — jump to a saved page
- **History** — find something you visited before
- **Actions** — browser commands like "New Tab" or "Clear History"

Results are ranked by relevance. Arrow keys to navigate, Enter to select, Escape to close.

## Actions

When the input is empty, the palette shows available actions:

| Action | What It Does |
|--------|-------------|
| New Tab | Opens a new tab |
| Close Tab | Closes the current tab |
| Toggle Compact | Toggles compact mode |
| Toggle Sidebar | Shows/hides the sidebar |
| Clear History | Wipes all browsing history |
| History | Opens the history panel |
| Bookmark | Bookmarks the current page |

## Shortcut

| Action | Key |
|--------|-----|
| Open/Close | `Ctrl+K` |
| Navigate | Arrow keys |
| Select | `Enter` |
| Close | `Escape` |
