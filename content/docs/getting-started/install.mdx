## Download

Grab the latest release from [GitHub Releases](https://github.com/visualstudioblyat/bushido/releases).

Bushido runs on Windows. macOS and Linux builds are coming.

## Requirements

- Windows 10 or later
- WebView2 runtime (ships with Windows 11, auto-installs on Windows 10)
